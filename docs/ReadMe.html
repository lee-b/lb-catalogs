Kintassa Catalogs - A Wordpress Plugin

Introduction

This plugin provides an easy, intuitive interface for creating catalogs. Features include:

  • Multiple Catalogs
  • Multiple Entries per catalog
  • Images, descriptions, titles, and links in each catalog entry
  • Images automatically scaleable to suit the styling of each page, on a per-page basis.
  • Links only used if provided
  • Plugin architecture for new/alternative rendering methods

Shortcodes and tags

Render a Catalog in post/page content using the kintassa_catalog shortcode, as follows:

[kintassa_catalog id="1" image_width="200" image_height="200"]

This will render the catalog from the admin that has the given id, scaling any images in the entries to fit the given dimensions.

In php code, use the kintassa_catalog tag/function, as follows:

<?php echo kintassa_catalog(1, image_width=200, image_height=200); >
